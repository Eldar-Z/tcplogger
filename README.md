# TCP Logger

Программа на Python, которая собирает информацию о трафике и количестве подключений.

## Установка

```bash
git clone https://github.com/Eldar-Z/tcplogger
```

## Использование

Используя make:
```bash
запуск программы--
make run

запуск тестирования--
make run
```
